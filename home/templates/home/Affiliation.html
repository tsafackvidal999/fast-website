Qu'est ce que l'affiliation?

Il s'agit d'une technique optimisation des revenus utilisée par certaines entreprises pour accroître leur rendement en développent des partenariats avec des personnes extérieurs qui leurs apportent des clients et reçoivent des commissions sur les achats de ces derniers.

comment ça marche dans notre cas?

  1. Créez votre compte affilié en remplissant le formulaire ci-dessous
    1. Choisissez votre votre code affilié (nous vous recommandons de choisir quelque chose de simple et facile à retenir)
    2. Définissez un nombre de jours gratuit (Cela encourage les utilisateurs à entrer votre code affilié. NB: ce nombre de jours que vous octroyez sera déduit sur vos commissions au premier achat(uniquement) de l'utilisateur. Vous pouvez mettre 0 jour si vous le souhaitez)
  2. Publicité
    1. Faites la promotion de votre code promo et du logiciel en invitant les utilisateurs à faire leur premier achat avec votre code promo pour avoir des jours gratuits.
    2. Les utilisateurs font leurs achats et c'est dans la poche. Vous aurez l'état de vos revenus sur votre tableau de bord

En souscrivant au programme d'affillition, vous approuvrez avoir lus et accepté les condictions générales d'Affiliation
